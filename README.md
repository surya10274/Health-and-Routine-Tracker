# Health & Routine Tracker (Python Console App)

## 📌 Project Description
This is a simple console-based Health & Routine Tracker developed using Python. It helps users track their daily health habits such as sleep hours, water intake, and personal habits.

---

## 🎯 Objective
The objective of this project is to create a simple and effective tool that allows users to monitor and improve their daily routines using basic Python programming concepts.

---

## ❗ Problem Statement
Many individuals, especially students, struggle with maintaining a healthy routine. They often lack awareness of their daily habits such as sleep and hydration. This project aims to solve this problem by providing an easy way to track these activities.

---

## 💡 Solution
The application provides a menu-driven interface where users can:
- Enter sleep hours
- Track water intake
- Add daily habits
- View saved data

All data is stored locally in a JSON file.

---

## 🛠️ Technologies Used
- Python
- JSON (for data storage)
